# VOA慢速英语 VOA slow English

VOA慢速英语APP for flutter。主要实现了音频和视频两个tab，以及播放详情页。

目前出于开发初级阶段，只提供基础的播放能力，暂未支持以下功能：
登录、口语测评、背单词、单词查询、收藏功能。
完整后台播放、视频下载等。

部分接口（例如每日一词）需要leancloud所需的id和sign，请自行APP抓包获取。
本项目仅供学习研究用途，遵循MIT协议。
